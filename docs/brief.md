# Viking Media — Project Brief

## Purpose
An AI-driven content pipeline built as a portfolio demonstration for corporate 
clients and employers looking to automate content creation. The project shows 
the full arc from AI-assisted (human edited) to automated publication, with 
the editorial feedback loop as the core differentiator.

## Target audience
- In-house content/comms teams at fintechs, banks, payment processors
- Agencies offering content services to financial sector clients
- Employers in AI content tooling or editorial automation

## What makes this distinct
The pipeline is not just a generation demo. It shows:
1. Prompt engineering discipline (versioned, beat-specific prompts)
2. Editorial feedback loop (diffs and scores drive systematic prompt refinement)
3. A demonstrable graduation from human-edited to automated output
4. Domain expertise in a specialist financial beat

## Owner background
Nick is a financial journalist, copywriter and editor based in Gateshead. 
He has copywriting experience for Klarna's B2B payments unit and other 
open banking/payments players, and AI evaluation experience at Contextual AI 
(document extraction for financial clients). This project combines both threads.
